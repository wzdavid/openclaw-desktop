## 🚀 使用示例

### 自适应优化
```python
from ai_humanizer_cn import Humanizer

h = Humanizer()

# 自动识别并优化
text = "本文介绍了一种新的实验方法"
result = h.humanize(text)  # 自动选择学术风格
# 输出："本文提出了一种创新性实验方法"
```

### 语境识别
```python
context = h.detect_context(text)
print(context)
# 输出：{
#   "domain": "academic",
#   "audience": "professional",
#   "purpose": "informative",
#   "tone": "neutral"
# }
```

### 长文本处理
```python
long_text = "段落 1\n\n段落 2\n\n段落 3" * 10
result = h.humanize_long_text(long_text, style="blog")
# 自动分块处理，保证连贯性
```

### 上下文感知
```python
h.humanize("我们做了实验")    # → "我们开展了实验"
h.humanize("我分析了数据")    # → "我们分析了数据" (代词一致)
```
