## 🔧 安装方式

### 方式 1：GitHub 安装

```bash
git clone https://github.com/小马 🐴/ai-humanizer-cn.git
cd ai-humanizer-cn
openclaw plugins install -l .
```

### 方式 2：ClawHub 安装

```bash
clawhub install ai-humanizer-cn
```

### 方式 3：手动部署

```bash
# 复制技能文件
cp -r ai-humanizer-cn /root/.openclaw/workspace/skills/

# 重启 OpenClaw
openclaw gateway restart
```
